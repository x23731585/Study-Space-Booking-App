# Study-Space-Booking-App
Java Application that allows users to search, book and manage study spaces.

Launch the application.
Enter a username and password to log in.
View available study spaces in the table.
Use the search field to find a specific study space.
Select a study space from the table.
Choose a date and time slot from the dropdown menus.
Click “Book Selected Space” to create a booking.
Click “Cancel Booking” to remove the most recent booking.
Booking details will be displayed in the booking details section.
